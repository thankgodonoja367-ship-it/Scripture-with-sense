# Scripture With Sense

Real questions. Honest answers. Helping young Christians understand truth and live it.

## About
This project is a simple platform where people can ask anonymous questions and get real, practical answers based on Christian truth.

## Features
- Scripture breakdowns
- Anonymous question submission
- Real-life answers
- Relationship guidance for teenagers

## Goal
To create a space where young Christians stop pretending and start growing through truth.

## Built With
- HTML
- CSS
- JavaScript
